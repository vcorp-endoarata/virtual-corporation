## 採用判断の補足: なぜ単体スコア最高の P1 を選ばなかったか

スコアカード上は P1 (タスク分解 SaaS 単体) が 78 点で最高。
しかし **CEO 判断として P2 ハイブリッドを採用** した。理由:

1. **代表者の強み活用度**: P1 単体は AI スキルしか活かせない。P2 ハイブリッドは「19歳・留年3年・ADHD・トゥレット・AI 趣味」の 5 強み全てを活かせる。
2. **比較競合の実態**: P1 単体は ChatGPT (月 3,000 円) と Goblin Tools (英語・無料) で代替可能。月 980 円課金が成立しにくい。
3. **当事者性の希少価値**: 「19 歳・留年 3 年・発達特性当事者が AI で復帰した」という人格は世界に 1 つしかない。これを活かすには P2 軸が必要。
4. **客単価**: P2 = 1,980 円のため、MRR 5 万円達成に必要な人数が **半分 (51人 → 26人)**。少人数の濃いコミュニティで成立する。
5. **ピボット可能性**: P2 ハイブリッドが失敗しても、コードはほぼそのまま「P1 単体」に縮退できる (= 実質的にロスが少ない)。

スコアカードは「機能の市場性」を測ったが、CEO 判断は「**機能 × 創業者の文脈**」で行われる。
今回はその両者を見て、スコア 2 位の方向 (P2) を採用した。
