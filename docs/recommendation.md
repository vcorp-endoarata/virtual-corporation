# V-Corp 推奨プロダクト + 14日で公開する計画

> 結論: **「留年・不登校・発達特性当事者のための AI 学習伴走 SaaS『ひとつ』」を 14日でデプロイし、決済受付まで開く**。
> Day 7 までに「今日やる 1 つ」のコア機能、Day 14 までにタスク分解機能を追加。

---

## 1. プロダクト定義 (1ページ仕様書)

### 1.1 名前と URL

- **プロダクト名**: **「ひとつ」** (Hitotsu)
- **プロダクト URL**: **https://hitotsu.v-corp.inc** (親ブランド .inc のサブドメイン)
- **親ブランド**: V-Corp
- **親ブランド URL**: **https://v-corp.inc** (V-Corp Studio LP)

→ ドメインは既に所有済み。Day 0 で取得する必要はない。

### 1.2 1行説明

> 今日やる 1 つだけ、AI が決めてくれる。

### 1.3 ターゲットユーザー

**プライマリ**: 中高生〜20代の **留年・不登校・通信制・発達特性** 当事者。
**セカンダリ**: 子に課金してあげたい **親** (進学塾・通信制サポート校より安い)。

具体的なペルソナ:
- 「やらなきゃいけないこと」が積み重なりすぎて、何から手を付けていいか分からず固まる
- スタディサプリ等の汎用カリキュラムは「健常者前提」で挫折経験あり
- ChatGPT は使えるが、毎回プロンプトを書く労力で続かない
- 人とのやりとり (家庭教師・コーチング) は症状的・心理的にしんどい

### 1.4 コア機能 (これだけ作る・これだけ売る)

| 機能 | 説明 | フェーズ |
|---|---|---|
| 状況入力 | 現状・目標・1日に使える時間を最初に登録 | Day 1-3 |
| **今日やる 1 つだけ** | 毎朝、AI が **1 つだけ** タスクを表示。それ以外は見せない | Day 1-7 (★コア) |
| 完了チェック | チェックで翌日のタスクが再計算される | Day 1-7 |
| 進捗ログ | 過去の「できた」を蓄積。続けるほど価値が上がる | Day 1-7 |
| **5 分タスクに分解** | 表示された「1 つ」がまだ重い時、AI が更に細かく分解 | Day 8-14 |
| 励ましトーン | 「動けないのはおかしくない」というコピー全般 | 全期間 |

**作らないもの (重要)**:
- ❌ コミュニティ・チャット (運営が消耗する)
- ❌ 友達招待・SNS 連携
- ❌ ガミフィケーション (ポイント・レベル・ランキング)
- ❌ カレンダー・通知 (鬱トリガーになる人がいる)
- ❌ 学習効果の保証・成績アップを謳う表現 (景表法・教育サービス化を回避)
- ❌ モバイルアプリ (まずは PWA で十分)

### 1.5 価格

| プラン | 月額 | 制限 |
|---|---|---|
| Free | 0円 | 7日間試用 / その後は閲覧のみ |
| Pro | **1,980円/月** | 無制限 + 履歴永続保存 + タスク分解機能 |

**MRR 5 万円ライン = 26 人**。X 案 (980円, 51 人) より人数が半分で済む。

## 2. 技術スタック (1日で全部動く構成)

| レイヤ | 技術 | 理由 |
|---|---|---|
| フロント | Next.js 15 (App Router) + Tailwind | Claude Code/v0 が一番得意 |
| 認証 | Supabase Auth (Google/メール) | 5分でセットアップ可 |
| DB | Supabase Postgres | 無料枠で十分 |
| 課金 | Stripe (Subscriptions) | 国内個人事業主でも対応 |
| LLM | Anthropic Claude API (Haiku) | 速くて安い |
| ホスティング | Vercel (Hobby プラン) | 無料 |
| 解析 | Plausible or Vercel Analytics | プライバシーフレンドリー |

**月次コスト見込み (ユーザー 50人時点)**:
- ドメイン (アクティブ3本のみ計上 / .inc + .fun + .tokyo): 〜1,000円/月相当
- Supabase: 0円 (Free 枠内)
- Vercel: 0円 (Hobby)
- Stripe: 売上の 3.6%
- Anthropic API (Haiku, 学習用途で多め): 〜4,000円/月
- **合計: 〜5,000円/月**

1,980円 × 50人 = 99,000円から、約 5%差し引かれる。**粗利 90%前後**。

**ドメイン年次コストの注意**: .inc TLD は更新料が高い (年 8万円〜10万円台のレジストラあり)。所有 18本の維持費は年 3-5万円規模になっている可能性が高い。**Day 90 のレビューで使っていない 12本以上は更新しない** 方針 (詳細は §10)。

## 4. 14日間の自分への約束

| 項目 | 約束 |
|---|---|
| 完璧主義の禁止 | 「もうちょっと作ってから公開する」は禁句 |
| 比較の禁止 | 他の SaaS と比べて落ち込まない |

| 公開する勇気 | 雑でも Day 7 に必ず告知する |
| 機能制限を守る | Day 7 まではタスク分解機能を作らない (Day 8 以降に解禁) |

## 5. 詰まったときのプロトコル (ADHD用)

| 状況 | やること |
|---|---|
| 30分以上 1つのバグで詰まった | Claude Code/ChatGPT に「丸投げ」する |
| やる気が出ない | この自分のサービスに「やる気が出ない」と入力して使う (本気) |
| 意味を見失った | README.md を読み返す。「自分が一番欲しいもの」を作っている |
| トゥレットの調子が悪い | その日は休む。別日に取り戻す前提でスケジュールに余裕を持たせている |
| 親や周囲に否定された | **無視して進める**。1人目の有料ユーザーが正解を教えてくれる |

## 6. お金まわりの実務 (19歳でも気にすること)

| 項目 | 対応 |
|---|---|
| 開業届 | 任意。年商 48万円超えるあたりで税務署へ提出推奨 |
| 確定申告 | 売上が出たら必須。会計ソフト (freee/マネーフォワード) は最初は無料枠で OK |
| 親の扶養 | 年収 103万円 (給与) / 48万円 (事業所得) 超えると外れる。事前に親と相談 |
| Stripe 契約 | 18歳以上なので個人で契約可 |
| 銀行口座 | 屋号付き口座は事業実績が必要。個人口座でスタートで OK |
| 消費税 | 売上 1,000万円超で課税事業者。当面気にしない |
| 賠償責任保険 | 月 1,000円程度のフリーランス向け保険 (FREENANCE 等)。Day 30 に検討 |
