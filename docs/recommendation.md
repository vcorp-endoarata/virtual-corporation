# V-Corp 推奨プロダクト + 7日で公開する計画

> 結論: **「ADHD のためのタスク分解 SaaS」を 7日でデプロイし、決済受付まで開く**。

---

## 1. プロダクト定義 (1ページ仕様書)

### 1.1 名前 (仮)

**「5min」** または **「ぶんかい」** または **「Splitr」**
→ Day 0 に1つ決める。悩みすぎない。後で変えられる。

### 1.2 1行説明

> ADHD の脳でも動ける、5分タスクに分解してくれる AI。

### 1.3 ターゲットユーザー

- ADHD・発達特性のある中高生〜社会人
- 「やらなきゃ」が頭の中で巨大ブロックになって動けない人
- 既存 TODO アプリ (Todoist, TickTick) では効かない人

### 1.4 コア機能 (これだけ作る・これだけ売る)

| 機能 | 説明 |
|---|---|
| 入力 | 「やらなきゃいけないこと」を1行で書く |
| 分解 | AI が 5分以内でできる極小タスクに自動分解 |
| 表示 | チェックリスト形式で1個ずつ表示 |
| 完了演出 | チェックすると軽いアニメ + 励まし文 |
| 履歴 | 過去の分解履歴を保存 |

**作らないもの (重要)**:
- ❌ コミュニティ機能
- ❌ 友達招待
- ❌ ガミフィケーション (ポイント・レベル)
- ❌ チーム機能
- ❌ カレンダー連携
- ❌ モバイルアプリ (まずは PWA でOK)

### 1.5 価格

| プラン | 月額 | 制限 |
|---|---|---|
| Free | 0円 | 月 10回まで分解可能 |
| Pro | **980円/月** | 無制限 + 履歴永続保存 |

## 2. 技術スタック (1日で全部動く構成)

| レイヤ | 技術 | 理由 |
|---|---|---|
| フロント | Next.js 15 (App Router) + Tailwind | Claude Code/v0 が一番得意 |
| 認証 | Supabase Auth (Google/メール) | 5分でセットアップ可 |
| DB | Supabase Postgres | 無料枠で十分 |
| 課金 | Stripe (Subscriptions) | 国内個人事業主でも対応 |
| LLM | Anthropic Claude API (Haiku) | 速くて安い |
| ホスティング | Vercel (Hobby プラン) | 無料 |
| 解析 | Plausible or Vercel Analytics | プライバシーフレンドリー |

**月次コスト見込み (ユーザー 50人時点)**:
- ドメイン: 125円/月
- Supabase: 0円 (Free 枠内)
- Vercel: 0円 (Hobby)
- Stripe: 売上の 3.6%
- Anthropic API (Haiku): 〜2,000円/月
- **合計: 〜2,500円/月**

980円 × 50人 = 49,000円から、約 5%差し引かれる。**粗利 90%超**。

## 3. 7日プラン (ADHD-friendly)

各日の **「ここまで動けば OK の最小ライン」** を明記。完璧を目指さない。

### Day 0 (今日 / 30分)

- [ ] サービス名を1つ決める (10分。悩んだら「5min」で確定)
- [ ] ドメイン取得 (5min.app, bunkai.app, splitr.jp など 1,500円程度)
- [ ] GitHub に空リポジトリ作成
- [ ] Vercel と Supabase に新規プロジェクト作成

**完了条件**: ドメインを所有し、Vercel に "Hello World" がデプロイされている。

---

### Day 1 (3時間)

- [ ] Next.js プロジェクトを Claude Code でスキャフォールド
- [ ] Supabase Auth (Google ログイン) を実装
- [ ] ダッシュボード画面に「ログインした人の名前」を出すだけ

**完了条件**: ログインできる。それだけ。デザイン気にしない。

**詰まったら**: 30分以上詰まったら Claude Code に「これが動かない」とそのまま投げる。自分で考え続けない。

---

### Day 2 (3時間)

- [ ] 入力フォームを作る (1行のテキストエリア + 送信ボタン)
- [ ] Anthropic Claude Haiku API に接続
- [ ] プロンプトを書く: 「以下のタスクを 5分以内でできる極小タスクに分解してください。各タスクは具体的・行動可能で、認知負荷が低いように書いてください。出力は JSON 配列のみ」
- [ ] 結果を画面に箇条書きで表示

**完了条件**: 「部屋を片付ける」と入力すると 5-10個の小タスクが返ってくる。

---

### Day 3 (3時間)

- [ ] 分解結果をチェックリスト UI にする (チェックボックス + ストライク)
- [ ] チェック時の演出 (色変化 + 「できた!」の小さなトースト)
- [ ] 履歴を Supabase に保存
- [ ] 履歴一覧画面

**完了条件**: 1日使い続けられる UX になっている。**自分が使いたいと思える** こと。

---

### Day 4 (3時間)

- [ ] Stripe アカウント作成・本人確認
- [ ] Stripe で「月額 980円」のプロダクトを作成
- [ ] Stripe Checkout を実装 (Pro プランへの導線)
- [ ] Webhook で Supabase の users テーブルに `is_pro` フラグを更新
- [ ] Free プランでは月 10回でブロックする処理

**完了条件**: 自分のクレカで自分のサービスにサブスク登録 → ちゃんと Pro になる。

---

### Day 5 (3時間)

- [ ] ランディングページ (1ページ)
  - ヒーローコピー: 「ADHD の脳でも動ける、5分タスクに分解する AI」
  - 動画 GIF (自分で実際に使ってみる動画 30秒)
  - 価格表
  - FAQ 5個
  - フッターに運営者情報・特商法表記
- [ ] 特商法・利用規約・プライバシーポリシー (テンプレ + AI で生成)

**完了条件**: 知らない人が読んで「使ってみよう」と思えるレベル。

**注意**: 特商法表記は **個人事業主でも住所・氏名の記載が必要**。バーチャルオフィス利用 (月990円〜) を検討。

---

### Day 6 (3時間)

- [ ] 自分で 5回以上使い倒す。バグを潰す。
- [ ] 簡単な onboarding (初回ログイン時の3ステップ説明)
- [ ] エラー時の表示を整える
- [ ] スマホでの表示確認

**完了条件**: 自分が「これなら他人に勧められる」と思える状態。

---

### Day 7 (2時間)

- [ ] X で告知ポスト
  - 内容例: 「19歳・留年3年・ADHD・トゥレットの自分が、自分のために作った。\n『やらなきゃ』が動かなくなる人に。\n月10回まで無料。\n[URL]」
- [ ] note に開発記録 (1記事)
- [ ] Product Hunt の Submission 準備 (即時投稿でなくてOK)
- [ ] ADHD 関連のハッシュタグでフォロワー外にも見せる

**完了条件**: 1人以上に使ってもらえている。**有料転換は焦らない**。

## 5. 詰まったときのプロトコル (ADHD用)

| 状況 | やること |
|---|---|
| 30分以上 1つのバグで詰まった | Claude Code/ChatGPT に「丸投げ」する |
| やる気が出ない | この自分のサービスに「やる気が出ない」と入力して使う (本気) |
| 意味を見失った | README.md を読み返す。「自分が一番欲しいもの」を作っている |
| トゥレットの調子が悪い | その日は休む。別日に取り戻す前提でスケジュールに余裕を持たせている |
| 親や周囲に否定された | **無視して進める**。1人目の有料ユーザーが正解を教えてくれる |

## 6. お金まわりの実務 (19歳でも気にすること)

| 項目 | 対応 |
|---|---|
| 開業届 | 任意。年商 48万円超えるあたりで税務署へ提出推奨 |
| 確定申告 | 売上が出たら必須。会計ソフト (freee/マネーフォワード) は最初は無料枠で OK |
| 親の扶養 | 年収 103万円 (給与) / 48万円 (事業所得) 超えると外れる。事前に親と相談 |
| Stripe 契約 | 18歳以上なので個人で契約可 |
| 銀行口座 | 屋号付き口座は事業実績が必要。個人口座でスタートで OK |
| 消費税 | 売上 1,000万円超で課税事業者。当面気にしない |
| 賠償責任保険 | 月 1,000円程度のフリーランス向け保険 (FREENANCE 等)。Day 30 に検討 |
