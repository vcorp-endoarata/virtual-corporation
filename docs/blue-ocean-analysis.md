# V-Corp ブルーオーシャン分析 (個人 webapp サブスク版)

> 19歳・ソロ・AI 趣味・ADHD+トゥレット という前提で、
> ウェブアプリ + 月額サブスクで成立する領域を再構成した分析。

---

## 1. 評価の前提

### 1.1 「成立する」の定義 (現実的に)

| 軸 | 合格ライン |
|---|---|
| 月額単価 | 500-2,000円 (B2C 上限。これ以上は B2B になり高校生不利) |
| 必要ユーザー数 | 月 5,000円なら 5-10人, 月 30万円なら 200-500人 |
| 個人で開発可能 | Next.js + Supabase + Stripe + LLM API で完結すること |
| 完全非同期 | チャット返信のみ。電話・Zoom 不要 |
| 当事者性 | あなたが「自分が一番欲しい」と言えるテーマであること |

### 1.2 除外する webapp 領域

これらは**個人の高校生が手を出すと負ける** か、相性が悪いので除外:

- ❌ **B2B SaaS** (法人契約・請求書発行・SLA を求められる)
- ❌ **マッチング系** (両面 PMF が必要、キャッシュフロー死ぬ)
- ❌ **金融・医療・法務** (規制で個人参入不可 or 賠償リスク)
- ❌ **動画・画像生成のラッパー** (大手が無料化して焼き尽くされた)
- ❌ **ChatGPT そのもののクローン** (差別化不能)
- ❌ **ハードウェア連動 / IoT** (ハード調達が地獄)
- ❌ **コミュニティ + AI ボット**程度 (Discord で十分。webapp である必要がない)

## 2. 候補プロダクト 6件

すべて **「自分が当事者として欲しいもの」** をベースにしている。これがソロ起業の最大の武器。

---

### 候補 P1: **「やる気が出ない」を5分タスクに分解する AI** (タスク分解 SaaS)

**痛み**: ADHD/不登校/留年経験者は「宿題やらなきゃ」「片付けなきゃ」が脳内で巨大ブロックになり動けない。
**機能**: ユーザーがやることを1行入力 → AI が **5分でできる極小タスク** に分解 → チェックリストで一個ずつ消せる → 完了で軽い演出。
**月額**: 980円 (無料プランは月10回まで)

**競合**:
- Goblin Tools (英語) — 海外で月数万人利用、日本語対応甘い
- 通常の TODO アプリ (Todoist など) — AI 分解機能は付いているが ADHD 特化ではない
- 国内専業 ADHD 向け SaaS = **ほぼゼロ**

**当事者性**: ★★★★★ (あなた自身が一番欲しい)
**競合密度**: 低 ◎
**実装難度**: 1機能で成立。1週間で MVP 可。

---

### 候補 P2: **「留年・不登校から復帰したい人」のための AI 個別学習計画**

**痛み**: 留年・不登校経験者は「どこから手を付けていいか」が壊滅的に分からない。
**機能**: 現在の状況・目標・1日に使える時間を入力 → AI が**今日やること1つだけ** を毎朝出す。進捗を入れると翌日が再計算される。
**月額**: 1,980円

**競合**:
- スタサプ・進研ゼミ — 健常者前提のカリキュラム
- 通信制高校サポート校 — リアル教室、個人で代替不可
- 「留年経験者 × AI 学習」専業 = **ゼロ**

**当事者性**: ★★★★★ (留年3年の経験そのものが商品)
**競合密度**: 極小 ◎◎
**実装難度**: コア機能はシンプル。継続率を上げる工夫が肝。

---

### 候補 P3: **「AI ヘビーユーザー向け プロンプト/エージェント個人ワークスペース」**

**痛み**: ChatGPT のカスタム指示、Claude Project、Gemini Gem、自作プロンプトが**散らばって管理不能**。
**機能**: 全部を1画面で一元管理。タグ・フォルダ・バージョン管理・公開/非公開切替。複数 LLM を切り替えて呼び出せる。
**月額**: 1,500円

**競合**:
- PromptHub, FlowGPT (海外) — 日本語特化はない
- Notion + テンプレ自作 — ハイレベル層は自前で作っている
- 国内専業 = ほぼなし

**当事者性**: ★★★★ (あなたが毎日困っているはず)
**競合密度**: 中
**実装難度**: 中。LLM API 連携が複数必要。

---

### 候補 P4: **「家族・先生・親戚に説明できない自分」を翻訳する AI**

**痛み**: 発達特性のある人が「親に学校行きたくない理由を伝える」「医者に症状を説明する」のがしんどい。
**機能**: 自分の状況を雑にチャット入力 → AI が**伝える相手別に整える** (親宛/医者宛/担任宛/X 投稿用)。テンプレ化された定型文ライブラリ付き。
**月額**: 980円

**競合**: 一般 AI チャット (ChatGPT) で代替可能だが、「相手別テンプレ」「履歴保存」「定型文ライブラリ」は専業の方が便利。
**当事者性**: ★★★★★
**競合密度**: 極小 ◎◎ (但し「便利だが必須ではない」と評価されるリスクあり)
**実装難度**: 低。MVP は3日で出せる。

---

### 候補 P5: **「チック/トゥレット 自己観察ログ + AI 分析」**

**痛み**: チック持ちは「いつ・なぜ出やすいか」を把握しきれていない。記録すると傾向が見えるがアプリがない。
**機能**: 1日数回の軽い入力 (チックの強さ・場所・直前の出来事) → 月次で AI が**トリガー候補レポート** を返す。
**月額**: 980円

**競合**: ほぼゼロ。但し医療領域で慎重さが必要。
**当事者性**: ★★★★★
**競合密度**: 極小 ◎◎
**実装難度**: 低-中
**リスク**: 「治療効果がある」と書くと薬機法・医療広告ガイドラインに抵触。**「自己観察の記録ツール」と明確に位置付けること**。

---

### 候補 P6: **「19歳の留年生が AI で生きるドキュメンタリー」サブスク note**

**痛み**: 同世代の進路に悩む人にとって「リアルな試行錯誤」を見たい需要は確実にある。
**機能**: webapp ではなく note 有料マガジン (月500円)。MRR 構造としてはサブスク同等。
**月額**: 500円

**位置付け**: webapp ではないので除外候補だが、**P1-P5 の集客装置** として並行運用する価値がある。

## 5. ERRC グリッド (P1 タスク分解 を例に)

| 軸 | 業界の常識 | V-Corp の戦略 |
|---|---|---|
| **Eliminate** | 多機能・多画面の TODO アプリ | 1機能 1画面のみ |
| **Eliminate** | 健常者前提の UX | 心理的ハードルを下げる文言・演出 |
| **Reduce** | 月額 1,500円以上の高単価 | 980円 (高校生でも払える) |
| **Reduce** | 招待・コミュニティ機能 | ソロで完結 |
| **Raise** | AI 機能の精度 | 当事者の声を取り込む頻度を上げる |
| **Raise** | 完了時の「達成感」演出 | ドーパミン設計を最優先 |
| **Create** | — | 「ADHD のための」と明示する勇気 (大手はやれない) |

## 6. 結論

### 6.1 戦略推奨

**3階建て構造**で並行運用する:

```
┌─────────────────────────────────────────┐
│ 1F (集客): P6 note サブスク 月500円     │
│   = 自分の試行錯誤を発信、ファン獲得    │
└─────────────────────────────────────────┘
                  ↓ 動線
┌─────────────────────────────────────────┐
│ 2F (本命): P1 タスク分解 SaaS 月980円  │
│   = 1週間で公開、3ヶ月で MRR 5万円     │
└─────────────────────────────────────────┘
                  ↓ 拡張
┌─────────────────────────────────────────┐
│ 3F (拡張): P2 留年復帰学習 月1,980円   │
│   = P1 が安定したら着手 (3-6ヶ月後)    │
└─────────────────────────────────────────┘
```

### 6.2 まず作るのは P1 (タスク分解)

理由:

1. **1機能で成立** = MVP を 1週間で出せる (ADHD と相性が良い短サイクル)
2. **自分が一番欲しい** = 開発中も自分で使い続けられる = 継続動機が高い
3. **市場が証明済み** (Goblin Tools が海外で成立している = ニーズ実在)
4. **日本語 ADHD 特化は空白** (大手はこのターゲット名を出せない)
5. **失敗時のダメージが小さい** (1週間で見切りをつけられる)

詳細実行計画は [recommendation.md](recommendation.md) を参照。
