# V-Corp (Virtual Corporation)

> 個人事業主として運営する「仮想株式会社」。1人 + AI で会社規模のアウトプットを出すことを目的とする。

## ミッション

法人化せず、個人事業主のままで「企業並みの専門性・スピード・スケール」を実現する。
組織を持たない代わりに、AI エージェント・自動化・プラットフォームを総動員する。

## 制約条件 (Constraints)

| 項目 | 制約 |
|---|---|
| 人員 | 1名 (代表者) + AI |
| 初期投資 | 50万円以下を想定 |
| 月次固定費 | 5万円以下を想定 |
| 法人化 | しない (個人事業主) |
| 在庫 | 持たない (デジタル/サービス事業のみ) |
| 営業 | 直販・PLG・コミュニティ主体 (アウトバウンド組織を持たない) |
