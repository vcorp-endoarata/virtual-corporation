# よりそい / Yorisoi

> 発達障害に悩む人々の **安らぎの場**

ADHD・ASD・トゥレット症候群 など、発達障害を持つ人と、その家族・身近な人たちが、
比較せず、攻撃せず、ただ寄り添える場所を作ります。

## 設計原則

- **「いいね」ボタンを置かない** — 比較を生まない
- **「うなずき」「共感」「共有」** — 静かな反応のみ
- **当事者 / 家族 / 共有** — 必要に応じてスペースを分離
- **半匿名** — ニックネーム、本人特定不可
- **モデレーション** — 攻撃的発言は通報・対応 (人力 → AI 補助 → モデレーターチーム段階的)
- **写真/動画は同意ベース** — 本人確認なし、注意書きベース、AI 安全チェック

## ターゲット

- 発達障害当事者本人
- 家族・身近な人
- 支援者・専門家

## ステータス

- v0: LP + ウェイトリスト収集 (現在)
- v1: 認証 + 投稿 + うなずき + スペース分離 + 通報 (1-2週間)
- v2: 写真・動画 / 都道府県登録 / 課金 ¥300/月 (3-4週間)
- v3: ルールベース危機検知強化 / オフ会公式機能 (2-3ヶ月)
